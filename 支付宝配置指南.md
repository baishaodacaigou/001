# 支付宝开放平台配置详细指南

## 第一步：注册支付宝开放平台账号

1. 访问 [支付宝开放平台](https://open.alipay.com/)
2. 点击右上角"登录"，使用支付宝APP扫码登录
3. 首次登录需要注册为开发者，按照提示完成身份认证

## 第二步：创建小程序应用

1. 登录后进入控制台
2. 点击左侧"小程序" → "创建"
3. 填写应用信息：
   - 应用名称：肌肉人AI租赁
   - 应用类型：小程序
   - 应用类目：按需选择（建议选择"工具"或"电商"）
4. 点击"创建"，等待审核通过（通常几分钟）
5. 审核通过后，你会看到 **AppID**，复制保存！

## 第三步：配置接口加签方式（获取密钥）

这是最重要的一步，需要配置应用私钥和支付宝公钥。

### 3.1 下载支付宝密钥生成工具

1. 在应用详情页，找到"开发信息" → "接口加签方式"
2. 点击"设置"
3. 选择"公钥"模式
4. 点击"下载密钥生成工具"（根据你的系统选择Windows或Mac版本）

### 3.2 生成密钥

1. 解压并运行密钥生成工具
2. 选择密钥格式：**PKCS1（非JAVA适用）**
3. 选择密钥长度：**2048**
4. 点击"生成密钥"
5. 工具会自动生成：
   - **应用私钥**（保密，不要泄露！）
   - **应用公钥**（需要上传到支付宝）
6. 点击"打开密钥文件位置"，保存好这两个文件

### 3.3 上传应用公钥并获取支付宝公钥

1. 回到支付宝开放平台"接口加签方式"设置页面
2. 将刚才生成的**应用公钥**复制粘贴到输入框
3. 点击"确认上传"
4. 上传成功后，页面会显示**支付宝公钥**，复制保存！

## 第四步：开通所需产品

### 4.1 开通"小程序获取会员信息"

1. 在应用详情页，点击"产品绑定"
2. 搜索"小程序获取会员信息"
3. 点击"立即开通"
4. 按照提示完成签约（免费）

### 4.2 开通芝麻信用产品（重要！）

1. 在产品绑定页面，搜索"芝麻信用"
2. 找到"芝麻信用评分"或"芝麻信用分"相关产品
3. 点击"立即开通"
4. 填写相关信息并提交审核
5. 注意：芝麻信用产品可能需要企业资质才能开通

## 第五步：配置到项目中

现在你有了以下三个关键信息：

1. **AppID**：从第二步获取
2. **应用私钥**：从第三步生成
3. **支付宝公钥**：从第三步获取

打开 `server/config/alipay.js`，填入这些信息：

```javascript
module.exports = {
  appId: '这里填你的AppID',
  
  privateKey: `-----BEGIN RSA PRIVATE KEY-----
这里填你的应用私钥（完整的，包括-----BEGIN和-----END）
-----END RSA PRIVATE KEY-----`,
  
  publicKey: `-----BEGIN PUBLIC KEY-----
这里填支付宝公钥（完整的，包括-----BEGIN和-----END）
-----END PUBLIC KEY-----`,
  
  gatewayUrl: 'https://openapi.alipay.com/gateway.do',
  notifyUrl: 'https://www.jirouai.com/api/alipay/notify',
  returnUrl: 'https://www.jirouai.com/api/alipay/return'
};
```

## 常见问题

### Q: 芝麻信用产品审核不通过怎么办？
A: 如果暂时无法开通芝麻信用，系统会自动使用默认信用分650，不影响基本登录功能。

### Q: 密钥格式错误怎么办？
A: 确保私钥和公钥都包含完整的 `-----BEGIN ...-----` 和 `-----END ...-----` 标记。

### Q: 如何测试配置是否正确？
A: 部署后，可以使用测试authCode `test123` 测试登录流程。

## 重要提醒

⚠️ **安全警告**：
- 应用私钥绝对不要上传到公开代码仓库
- 不要将私钥分享给他人
- 建议在服务器上使用环境变量存储密钥
